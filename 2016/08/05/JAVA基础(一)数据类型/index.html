JAVA 数据类型介绍

现在每天的工作无时无刻不在使用 JAVA 码代码，但是对 JAVA 很多基础知识了解并不是很透彻，有时可能是错误的用法。这里计划归纳学习 JAVA 基础知识。首先要学习的就是 JAVA 的数据类型，我们接触最多应该就是不同的数据类型的定义和声明使用。

为什么需要数据类型

  • 数据类型可以通过代码暂时用来存储数据；

  • 数据类型指定了将要被存储到存储单元的数据类型

    定义

  • 数据类型是提供信息介绍的东东；

  • 定义了存储器可以容纳的存储单元范围和数据范围；
  • 在存储器中所有合法的数据操作可以被执行；
  • 当我们使用该数据类型的表达式会产生什么的结果；
  • JAVA 编程语言提供很多不同类别的数据类型；
  • 无论关键字给出的语义都将被视为「数据类型」。

不同的数据类型

可以分为两大类数据类型：

  • 原始数据类型（基础数据类型）
  • 引用数据类型（派生数据类型）

从第一幅图就可以看出 JAVA 一共原始数据类型有八种类型，引用数据类型有两种。下面我们来详细介绍它们以及它们的关键知识点。

  1. 原始数据类型
  • 原始数据类型是指编程语言定义好的数据类型和它们所表示的意思，原始数据类型是用来存储单一的数据值的。
  • 依据数据的类型和数据值的范围，将原始数据类型划分为八种数据类型。

1.1 整型类别

  • 这里整型类型不是专门指 int 这一专门的类型而是指一个大类。先说下这类的定义内容这类主要是指那些存储数字的数据类型，既包括了正数也包括了负数但不包括小数；
  • 在这个大类里我们有四个数据类型它们的主要区别是内存大小的不同；
  • 这四种数据类型都是用来存储数字的，但是它们存储的数字范围是不同的。JAVA 在整型这个大类中提供这四种数据类型的主要目的是使内存的使用更高效点。
  1. byte ：byte 占 8-bit 位数的有符号补整数据类型，因为是有符号的所以它的数据值的范围是 -128~127 可以表示 256 个数字。
  2. short ：short 占 16-bit 位数的有符号数据类型，跟 byte 一样它表示的范围在 -32768~32767 。
  3. int ：int 默认占 32-bit 位数有符号补整数据类型，因为有符号所以它的数值范围是 -2147483648~2147483647。在 JAVA 1.8 及更高版本 int 可以表示无符号整型，这样的它能表示的范围就可以达到 0 到 ，要实现 int 型来表示无符号 int 需要使用 Integer 类。
  4. long ；long 占 64-bit 有符号数据类型，表示数据的范围是 到 。同样跟 int 类型一样的也可以表示为 无符号类型。

1.2 浮点数类别

  • 这类数据类型可以用来存储一个小数形式的数字通过 +VE 或者 -VE 来表示。在浮点数类型中有两种不同数据范围的数据类型: float 和 double
  • 和前面的一样的 JAVA 通过这两种数据类型来表示不同数值范围来节约内存；
  • float 默认值是 0 ；
  • double 默认值是 0.0 。
  1. float ： 占用位数是 32-bit 位，如果你的程序要考虑节约内存的话，在同样的需求下（数据范围）建议使用 float类型。该类型不应该用于精确的数据，比如说货币。
  2. double ：占用的位数是 64-bit，这种数据类型很实用用于小数数字的默认选择类型。

1.3 字符类型

  • 这里类数据类型应该被用于存储单个字符，这个字符可以是字母、数字甚至是一个符号；
  • 该大类只包含一个数据类型并被作为字符类型；
  • 默认值是 「one space」
  1. char ：16-bit Unicode 数据类型，数据范围是 0-65535 也可以表示为 ‘\u0000’ 到 ‘\uffff’。

1.4 布尔类型

这种类型没什么好说的两个数据值 true false。

上图是原始数据类型的默认值。总结下原始数据类型的局限性，第一我们不能用在连续的存储单元存储多个值，第二个就是我们要面对两个问题。

  • 如果我们存储多个数值的话，比如 100 个数据，我们就要定义 100 个数据变量，所有这些变量的定义创建都是通过 JVM 存储在不同的存储单元中。如下图所示，因此在查找的时候会耗费很多时间。
  • 如果想通过网络向远程计算机传递数据使用数据类型就显得无法完成了，因为这会给网络传输增加压力同样为会在程序中增加好多行。
  1. 引用类型

引用类型的出现就是为了解决上述问题，通过一个变量将值存储在连续的存储单元中，这样就可以使用引用类型了（自己理解跟多个变量指向同一个存储单元）。因为引用类型是通过一个变量将值存储在连续的存储单元中，因此查询速度会比较快。而且可以通过一个而网络请求来传输数据。

引用数据类型有以下三种：

  1. Array
  2. Class
  3. Interface

double 和 float 的区别

前面的数据类型的介绍已经详细介绍了，二者的区别主要是在数据的精度上，更详细的区别如下：

  1. 范围
    float 和 double 范围是由指数的范围来决定的，float 表示的指数范围是 8 位，double 指数范围是 11 位。

float
1bit (符号位) + 8bits (指数位) + 23bits (尾数位)
double
1bit(符号位) + 11bits (指数位) + 52bits (尾数位)

  1. 精度
    从上面可以看出 float 和 double 最大的区别是指数位和尾数位，float 和 double 都是采用科学计数的方法，整数部分始终是 1 位，它是不变的不会对精度造成影响。

  2. 影响
    float 和 double 都是可以表示小数的，double 占用内存大运行速度慢，但是他表示的精度大，所以在不确定小数位数的情况下建议使用 double。

数据类型后缀

因为我们在定义 long 、float 、double 时后面都会增加 L(l) 、F(f) 、D(d) 其实两个都可以用只是 L 建议使用大写，因为 1 和 1 很像。
